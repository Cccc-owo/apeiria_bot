# 帮助系统

_✨ Apeiria 的图像化帮助菜单与插件详情插件 ✨_

</div>

## 📖 介绍

这是 Apeiria 的内置帮助插件，用于把当前机器人已经接入的功能整理成统一入口。

适合这些场景：

- 给普通用户提供统一的功能菜单入口
- 展示插件说明、命令别名与功能摘要
- 让主人快速查看某个插件的大致用途

## 💿 安装

<details open>
<summary>作为 Apeiria 内置插件使用</summary>

帮助系统为 Apeiria 内置插件，默认随框架加载，无需单独安装。

如果你关闭了此插件，可在项目插件配置中重新启用：

    apeiria.plugins.help

</details>

## ⚙️ 配置

在 Apeiria 的插件配置中可调整以下常用选项：

| 配置项 | 必填 | 默认值 | 说明 |
|:---:|:---:|:---:|:---:|
| `title` | 否 | `帮助菜单` | 帮助菜单标题 |
| `subtitle` | 否 | 空 | 帮助菜单副标题 |
| `accent_color` | 否 | `#4e96f7` | 菜单主色 |
| `show_builtin_cmds` | 否 | `false` | 是否显示内建与框架插件 |
| `plugin_blacklist` | 否 | `[]` | 从帮助菜单中隐藏的插件列表 |
| `admin_show_all` | 否 | `false` | 是否给主人展示完整帮助 |
| `expand_commands` | 否 | `false` | 是否展开显示命令 |
| `custom_templates` | 否 | `false` | 是否优先加载自定义模板 |
| `font_urls` / `font_family` | 否 | 空 | 字体相关设置 |
| `banner_image` / `header_logo` | 否 | 空 | 顶部横幅与标识图 |
| `footer_text` | 否 | 空 | 页脚文本 |

## 🎉 使用

### 指令表

| 指令 | 权限 | 需要@ | 范围 | 说明 |
|:---:|:---:|:---:|:---:|:---:|
| `/help` | 全部用户 | 否 | 群聊 / 私聊 | 查看帮助菜单 |
| `/help <插件名>` | 全部用户 | 否 | 群聊 / 私聊 | 查看指定插件详情 |
| `/help --all` | 主人 | 否 | 群聊 / 私聊 | 查看完整帮助 |

## 注意

- 帮助系统依赖 `apeiria.plugins.render`。
- 如果统一渲染服务不可用，帮助菜单将无法正常生成图片。
- 在项目功能尚未稳定时，可以先关闭此插件，待功能整理完成后再启用。
